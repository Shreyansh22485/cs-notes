# LC1092. Shortest Common Supersequence

[LeetCode Problem Link](https://leetcode.com/problems/shortest-common-supersequence/)

## Intuition & Approach
To find the shortest common supersequence (SCS) of two strings, we want to construct a sequence that contains both target strings as subsequences while minimizing its total length.

### Approach 1: Via Longest Common Subsequence (LCS)
- To minimize the length of the supersequence, we must maximize the characters that are shared between `text1` and `text2` in their original order. This is the **Longest Common Subsequence (LCS)** problem.
- The length of the SCS is given by $N + M - \text{LCS}$, where $N$ and $M$ are the lengths of `text1` and `text2`.
- We build the LCS table `dp` where `dp[i][j]` is the length of the LCS of prefixes `text1[0...i-1]` and `text2[0...j-1]`.
- We backtrack from `(n, m)` to `(0, 0)`:
  - If `text1[i-1] == text2[j-1]`, the character is part of the LCS. We prepend it and move diagonally up-left (`i--, j--`).
  - If they differ, we compare the LCS values `dp[i-1][j]` and `dp[i][j-1]`. We move in the direction of the larger LCS value, prepending the character we skip.

### Approach 2: Direct SCS Length State Tabulation
- We can define `dp[i][j]` directly as the length of the SCS of `s1[0...i-1]` and `s2[0...j-1]`.
- Base Cases: If one string is empty, the SCS is simply the other string:
  - `dp[i][0] = i` (for all $i$)
  - `dp[0][j] = j` (for all $j$)
- State Transitions:
  - If `s1[i-1] == s2[j-1]`, we include this character once: `dp[i][j] = 1 + dp[i-1][j-1]`.
  - If they differ, we take the minimum of including `s1[i-1]` or `s2[j-1]`: `dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1])`.
- Backtracking: We start at `(n, m)` and move backwards. If characters differ, we move in the direction of the smaller SCS value (`dp[i-1][j] < dp[i][j-1]` to include `s1[i-1]`, else `s2[j-1]`).

## Code Implementation

### Approach 1: Via Longest Common Subsequence (LCS)
```cpp
class Solution {
public:
    string shortestCommonSupersequence(string text1, string text2) {
        int n=text1.size();
        int m=text2.size();
        vector<vector<int>> dp(n+1,vector<int>(m+1,0));
        
        for(int ind1=1;ind1<=n;ind1++){
            for(int ind2=1;ind2<=m;ind2++){
                if(text1[ind1-1] == text2[ind2-1]) dp[ind1][ind2]=1+dp[ind1-1][ind2-1];
                else dp[ind1][ind2]=max(dp[ind1-1][ind2],dp[ind1][ind2-1]);
            }
        }
        
        int i=n;
        int j=m;
        string ans="";
        while(i>0 && j>0){
            if(text1[i-1]==text2[j-1]){
                ans=text1[i-1]+ans;
                i--;
                j--;
            }
            else if(dp[i-1][j]>dp[i][j-1]){
                ans=text1[i-1]+ans;
                i--;
            }
            else{
                ans=text2[j-1]+ans;
                j--;
            }
        }
        while(i>0){
                ans=text1[i-1]+ans;
                i--;
            
        }
        while(j>0){
            ans=text2[j-1]+ans;
                j--;
        }
        return ans;
    }
};
```

### Approach 2: Direct SCS Length State Tabulation
```cpp
class Solution {
public:
    int f(int n, int m, string& s1, string& s2, vector<vector<int>>& dp){
        if(n == 0 || m == 0) return n + m;
        if(dp[n][m] != -1) return dp[n][m];

        if(s1[n-1] == s2[m-1]) return dp[n][m] = 1 + f(n-1,m-1,s1,s2,dp);
        return dp[n][m] = 1 + min(f(n-1,m,s1,s2,dp),f(n,m-1,s1,s2,dp));
    }
    string shortestCommonSupersequence(string s1, string s2) {
        int n = s1.size();
        int m = s2.size();

        // vector<vector<int>> dp(n+1,vector<int>(m+1,-1));
        // return f(n,m,s1,s2,dp); // Start from n,m so that Recursion + Memo code is similar to bottom-up
        
        vector<vector<int>> dp(n+1,vector<int>(m+1,0)); // length of scs btw s1 of length n & s2 of length m

        for(int i=0;i<=n;i++){
            dp[i][0] = i;
        }
        for(int j=0;j<=m;j++){
            dp[0][j] = j;
        }

        for(int i=1;i<=n;i++){
            for(int j=1;j<=m;j++){
                if(s1[i-1] == s2[j-1]) dp[i][j] = 1 + dp[i-1][j-1];
                else dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1]);
            }
        }

        int i=n;
        int j=m;
        string ans="";
        while(i>0 && j>0){
            if(s1[i-1]==s2[j-1]){
                ans=s1[i-1]+ans;
                i--;
                j--;
            }
            else if(dp[i-1][j] < dp[i][j-1]){
                ans=s1[i-1]+ans;
                i--;
            }
            else{
                ans=s2[j-1]+ans;
                j--;
            }
        }
        while(i>0){
                ans=s1[i-1]+ans;
                i--;
            
        }
        while(j>0){
            ans=s2[j-1]+ans;
                j--;
        }
        return ans;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N \times M)$ for both approaches, as both construct an $N \times M$ DP table and perform a linear backtracking traversal.
- **Space Complexity**: $O(N \times M)$ for storing the DP state tables in both approaches.

---
**Key Takeaways**:
- **LCS-SCS Duality**: The Shortest Common Supersequence length is complementary to the Longest Common Subsequence length, given by $N + M - \text{LCS}$.
- **Direct State Formulation**: Defining DP states to match the target objective directly (SCS length) helps align the transition logic with the problem constraints, swapping the `max` operation in LCS for a `min` operation.
