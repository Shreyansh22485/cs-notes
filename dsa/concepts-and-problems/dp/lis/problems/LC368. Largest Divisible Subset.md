# LC368. Largest Divisible Subset

[LeetCode Problem Link](https://leetcode.com/problems/largest-divisible-subset/)

## Intuition & Approach
- We want to find the largest subset of `nums` such that every pair $(a, b)$ satisfies $a \pmod b == 0$ or $b \pmod a == 0$.
- Sort `nums` in ascending order. If $nums[i]$ divides $nums[j]$ (where $j < i$), then $nums[i]$ also divides all elements in the divisible subset ending at $nums[j]$.
- We define `dp[i]` as the size of the largest divisible subset ending at $nums[i]$, initialized to 1.
- We maintain `parent[i]` to store the predecessor index, allowing us to reconstruct the subset.
- For each $i$ from 0 to $N-1$ and $j$ from 0 to $i-1$:
  - If $nums[i] \pmod{nums[j]} == 0$ and `dp[j] + 1 > dp[i]`, update `dp[i] = dp[j] + 1` and `parent[i] = j`.
- Reconstruct the subset by tracing back from the index of the maximum `dp` value.

## Code Implementation
```cpp
class Solution {
public:
    vector<int> largestDivisibleSubset(vector<int>& arr) {
        int n=arr.size();
        sort(arr.begin(),arr.end());
        vector<int> dp(n,1);
        vector<int> hash(n);
        int maxi=0;
        int lastIndex=0;
        for(int i=0;i<n;i++){
            hash[i]=i;
            for(int prev=0;prev<i;prev++){
                if(arr[i]%arr[prev]==0  && 1+dp[prev]>dp[i]){
                    dp[i]=1+dp[prev];
                    hash[i]=prev;
                }
            }
            if(dp[i]>maxi){
                maxi=dp[i];
                lastIndex=i;
            }
        }
        vector<int> temp;
        temp.push_back(arr[lastIndex]);
        while(hash[lastIndex]!=lastIndex){
            lastIndex=hash[lastIndex];
            temp.push_back(arr[lastIndex]);
        }
        reverse(temp.begin(),temp.end());
        return temp;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N^2)$ to examine all pairs. Sorting takes $O(N \log N)$ time.
- **Space Complexity**: $O(N)$ for DP and parent tracking tables.

---
**Key Takeaways**:
- **Subsequence Reconstruction**: Maintain a `parent` pointer array parallel to the DP state array to reconstruct the optimal path after finding the maximum value.
