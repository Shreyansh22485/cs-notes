```cpp
class Solution {
public:
    vector<int> largestDivisibleSubset(vector<int>& arr) {
        int n=arr.size();
        sort(arr.begin(),arr.end());
        vector<int> dp(n,1);
        vector<int> hash(n);
        int maxi=0;
        int lastIndex=0;
        for(int i=0;i<n;i++){
            hash[i]=i;
            for(int prev=0;prev<i;prev++){
                if(arr[i]%arr[prev]==0  && 1+dp[prev]>dp[i]){
                    dp[i]=1+dp[prev];
                    hash[i]=prev;
                }
            }
            if(dp[i]>maxi){
                maxi=dp[i];
                lastIndex=i;
            }
        }
        vector<int> temp;
        temp.push_back(arr[lastIndex]);
        while(hash[lastIndex]!=lastIndex){
            lastIndex=hash[lastIndex];
            temp.push_back(arr[lastIndex]);
        }
        reverse(temp.begin(),temp.end());
        return temp;
    }
};
```
- Time Complexity: O(n^2) - We are using two nested loops to iterate through the array of numbers, where n is the number of elements in the array. The outer loop runs n times, and for each iteration of the outer loop, the inner loop can run up to n times in the worst case. Additionally, we are sorting the array at the beginning, which takes O(n log n) time. Therefore, the overall time complexity is O(n^2 + n log n), which simplifies to O(n^2).
- Space Complexity: O(n) - We are using two 1D arrays of size n, `dp` and `hash`, to store the length of the largest divisible subset ending at each index and the previous index in the subset, respectively. Additionally, we are using a temporary vector `temp` to store the final result, which can also take up to O(n) space in the worst case. Therefore, the overall space complexity is O(n).
