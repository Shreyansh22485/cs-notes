# LC1027. Longest Arithmetic Subsequence

[LeetCode Problem Link](https://leetcode.com/problems/longest-arithmetic-subsequence/)

## Intuition & Approach
- We want to find the length of the longest arithmetic subsequence in an array `nums`.
- We define `dp[i][diff]` as the length of the longest arithmetic subsequence ending at index $i$ with a common difference `diff`.
- For each index $i$ from 0 to $N-1$, and for each index $j$ from 0 to $i-1$:
  - Calculate the difference `diff = nums[i] - nums[j]`.
  - Transition: `dp[i][diff] = dp[j][diff] + 1` (defaulting to 2 if `dp[j][diff]` was 0).
- We track the maximum value across all states. Since difference values can be negative, we offset them by 500 or use a hash map.

## Code Implementation
```cpp
class Solution {
public:
    int longestArithSeqLength(vector<int>& nums) {
        int n = nums.size();
        vector<unordered_map<int, int>> dp(n);
        int maxLength = 0;

        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < i; ++j) {
                int diff = nums[i] - nums[j];
                dp[i][diff] = dp[j].count(diff) ? dp[j][diff] + 1 : 2;
                maxLength = max(maxLength, dp[i][diff]);
            }
        }

        return maxLength;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N^2)$ because we examine all pairs $(j, i)$.
- **Space Complexity**: $O(N \cdot \text{diff\_range})$ to store the DP table.

---
**Key Takeaways**:
- **Multi-Dimensional DP States**: When state transitions depend on a variable constraint (like the difference `diff`), adding that constraint as a state dimension (`dp[index][constraint]`) resolves the transition.
