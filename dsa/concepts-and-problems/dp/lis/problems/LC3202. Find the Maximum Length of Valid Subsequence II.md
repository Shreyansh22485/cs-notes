```cpp
class Solution {
public:
    int maximumLength(vector<int>& nums, int k) {
        int n = nums.size();
        vector<vector<int>> dp(k, vector<int>(n, 1)); // LIS for each mod
        int maxSub = 1;

        for(int i = 1; i < n; i++) {
            for(int j = 0; j < i; j++) {
                int mod = (nums[j] + nums[i]) % k;
                dp[mod][i] = max(dp[mod][i], 1 + dp[mod][j]);
                maxSub = max(maxSub, dp[mod][i]);
            }
        }


        return maxSub;

    }
};
```
- Time Complexity: O(n^2), where n is the length of the input array nums. We have two nested loops iterating through the array.
- Space Complexity: O(n*k), where k is the given integer. We are using a 2D vector to store the lengths of valid subsequences for each index and modulo value.
