```cpp
class Solution {
public:
    int climbStairs(int n) {

        vector<int> dp(n+1,0);
        int prev2=1;
        int prev1=1;
        int cur;
        for(int i=2;i<=n;i++){
            cur =prev1+prev2;
            prev2=prev1;
            prev1=cur;
        }
        return prev1;
    }
};
```
- Time Complexity: O(n) - We are iterating from 2 to n, performing constant time operations in each iteration.
- Space Complexity: O(1) - We are using a constant amount of space for the variables `prev2`, `prev1`, and `cur`, regardless of the input size n.
