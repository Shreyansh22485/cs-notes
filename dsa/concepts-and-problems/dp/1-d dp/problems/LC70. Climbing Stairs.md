# LC70. Climbing Stairs

[LeetCode Problem Link](https://leetcode.com/problems/climbing-stairs/)

## Intuition & Approach
- We want to reach the $N$-th step. At each step, we can climb either 1 or 2 stairs.
- To reach step $i$, we could have come from step $i-1$ (via a 1-step climb) or step $i-2$ (via a 2-step climb).
- Therefore, the recurrence is `dp[i] = dp[i-1] + dp[i-2]`, which matches the Fibonacci sequence structure. We initialize `prev2 = 1` and `prev1 = 1` and iterate up to $N$.

## Code Implementation
```cpp
class Solution {
public:
    int climbStairs(int n) {

        vector<int> dp(n+1,0);
        int prev2=1;
        int prev1=1;
        int cur;
        for(int i=2;i<=n;i++){
            cur =prev1+prev2;
            prev2=prev1;
            prev1=cur;
        }
        return prev1;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N)$ time.
- **Space Complexity**: $O(1)$ auxiliary space.

---
**Key Takeaways**:
- **Combinatoric Path Addition**: The total ways to reach a state is the sum of ways to reach all possible predecessor states.
