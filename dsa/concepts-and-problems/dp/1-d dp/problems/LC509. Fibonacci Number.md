```cpp
class Solution {
public:
    int fib(int n) {
        if(n <= 1) return n;
        int secondPrev = 0;
        int prev = 1;
        int cur  = 0;
        for(int i=2;i<=n;i++){
            cur = prev + secondPrev;
            secondPrev = prev;
            prev = cur;
        }
        return cur;
    }
};
```
- Time Complexity: O(n) - We are iterating from 2 to n, performing constant time operations in each iteration.
- Space Complexity: O(1) - We are using a constant amount of space for the variables `secondPrev`, `prev`, and `cur`, regardless of the input size n.
