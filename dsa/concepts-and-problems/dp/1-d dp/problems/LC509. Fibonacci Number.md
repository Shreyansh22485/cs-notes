# LC509. Fibonacci Number

[LeetCode Problem Link](https://leetcode.com/problems/fibonacci-number/)

## Intuition & Approach
- Compute the $N$-th Fibonacci number defined by $F(N) = F(N-1) + F(N-2)$ with base cases $F(0) = 0$ and $F(1) = 1$.
- We solve this using a bottom-up tabulation approach. For $O(1)$ space, we maintain `prev2 = 0` and `prev1 = 1` and compute `curr = prev1 + prev2` at each step up to $N$.

## Code Implementation
```cpp
class Solution {
public:
    int fib(int n) {
        if(n <= 1) return n;
        int secondPrev = 0;
        int prev = 1;
        int cur  = 0;
        for(int i=2;i<=n;i++){
            cur = prev + secondPrev;
            secondPrev = prev;
            prev = cur;
        }
        return cur;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N)$ time.
- **Space Complexity**: $O(1)$ auxiliary space.

---
**Key Takeaways**:
- **Bottom-Up State Induction**: The simplest example of state tabulation where a value is computed purely from its fixed preceding offsets.
