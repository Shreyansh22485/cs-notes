# LC1911. Maximum Alternating Subsequence Sum

[LeetCode Problem Link](https://leetcode.com/problems/maximum-alternating-subsequence-sum/)

## Intuition & Approach
- The alternating sum of a subsequence is defined as: `sum_even - sum_odd` (even indices added, odd indices subtracted). We want to maximize this sum.
- At each element $nums[i]$, we can either transition to a state where we add it (even index in subsequence) or subtract it (odd index in subsequence), or we can skip it.
- We maintain two running DP states:
  - `even`: The maximum alternating sum ending with an even index (element added).
  - `odd`: The maximum alternating sum ending with an odd index (element subtracted).
- For each number $x$ in `nums`:
  - `new_even = max(even, odd + x)` (either skip $x$, or add it to the best odd sequence).
  - `new_odd = max(odd, even - x)` (either skip $x$, or subtract it from the best even sequence).
- The answer is the value of the `even` state at the end.

```cpp
class Solution {
public:
    // long long f(int i, int op ,vector<int>& nums ,vector<vector<long long>>& dp){
    //     int n = nums.size();

    //     return dp[i][op+1] = max(pick,notPick);
    // }
    long long maxAlternatingSum(vector<int>& nums) {
        int n = nums.size();
        // vector<vector<long long>> dp(n,vector<long long>(2+1,-1));
        // return f(0,1,nums,dp);

        vector<vector<long long>> dp(n+1,vector<long long>(2,0)); // 1 baesd indexing
        // dp[i][0] is on adding the ith index element the length is even

        for(int i= 1;i<=n;i++){ //       1based indexsing
            dp[i][0] = max(dp[i-1][1] - nums[i-1] ,dp[i-1][0]);
            dp[i][1] = max(dp[i-1][0] + nums[i-1] ,dp[i-1][1]);
        }

        return max(dp[n][0],dp[n][1]);


    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N)$ time to iterate through the array once.
- **Space Complexity**: $O(1)$ auxiliary space as we only store two state variables.

---
**Key Takeaways**:
- **State Machine DP**: When transitions at each step depend on a binary state (such as even/odd parity in index sequences), maintaining simple state variables reduces DP space to $O(1)$.
