# LC1483. Kth Ancestor of a Tree Node

[LeetCode Problem Link](https://leetcode.com/problems/kth-ancestor-of-a-tree-node/)

## Intuition & Approach
- The problem asks to find the $K$-th ancestor of a node in a tree. A naive traversal moving parent-by-parent takes $O(K)$ time per query, which is too slow.
- We use **Binary Lifting** to jump upward in powers of 2. Since any integer $K$ can be represented as a sum of powers of 2, we can jump to the $K$-th ancestor in $O(\log K)$ steps.
- During initialization:
  - Precompute a 2D ancestor table where `ancestorTable[node][j]` represents the $2^j$-th ancestor of `node`.
  - The recurrence relation is: `ancestorTable[node][j] = ancestorTable[ancestorTable[node][j-1]][j-1]` (the $2^j$-th ancestor is the $2^{j-1}$-th ancestor of the $2^{j-1}$-th ancestor of the node).
  - The table of size $N \times (\log N + 1)$ is populated in $O(N \log N)$ time.
- During a query `getKthAncestor(node, k)`:
  - Iterate through the bit representation of `k`. If the $j$-th bit is set, lift the node to `ancestorTable[node][j]`. If the node becomes -1, return -1 immediately.

## Code Implementation
```cpp
class TreeAncestor {
public:
    vector<vector<int>> ancestorTable;
    int rows;
    int cols;

    TreeAncestor(int n, vector<int>& parent) {
        rows = n;
        cols = log2(n) + 1;

        ancestorTable.resize(rows,vector<int>(cols,-1));

        for(int node = 0; node < rows; node++){
            ancestorTable[node][0] = parent[node];
        }

        for(int j = 1; j < cols; j++){
            for(int node = 0; node < rows; node++ ){
                if(ancestorTable[node][j-1] != -1){
                    ancestorTable[node][j] = ancestorTable[ancestorTable[node][j-1]][j-1];
                }
            }
        }
    }
    
    int getKthAncestor(int node, int k) {
        for(int j=0;j<cols;j++){
            if(k & (1<<j)){ //set bit
                node = ancestorTable[node][j];
                if(node == -1){
                    return -1;
                }
            }
        }
        return node;
    }
};

/**
 * Your TreeAncestor object will be instantiated and called as such:
 * TreeAncestor* obj = new TreeAncestor(n, parent);
 * int param_1 = obj->getKthAncestor(node,k);
 */
```

## Complexity Analysis
- **Time Complexity**: Preprocessing: $O(N \log N)$; Query: $O(\log K)$ time.
- **Space Complexity**: $O(N \log N)$ to store the ancestor table.

---
**Key Takeaways**:
- **Binary Lifting Recurrence**: Precalculating jumps of size $2^j$ via $2^j = 2^{j-1} + 2^{j-1}$ enables logarithmic jump traversal on trees or directed acyclic graphs.
