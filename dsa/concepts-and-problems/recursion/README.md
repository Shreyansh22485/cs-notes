# Recursion Framework

1. Break the problem into smaller subproblems.
2. Call the same function recursively to solve the subproblems.
3. Combine the results of the subproblems to solve the original problem.

## Base Case
The base case is the condition that stops the recursion. It is the simplest instance of the problem that can be solved directly without making any further recursive calls.

## Recurrence Relation
Pattern or Rule that defines the relationship between the original problem and its subproblems.
