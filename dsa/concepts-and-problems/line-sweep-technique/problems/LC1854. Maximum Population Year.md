# LC1854. Maximum Population Year

[LeetCode Problem Link](https://leetcode.com/problems/maximum-population-year/)

## Intuition & Approach
- We are given logs of births and deaths: `[birth, death]`. We want to find the earliest year with the maximum population.
- We use a difference array of size 101 (representing years from 1950 to 2050 offset by 1950):
  - For each log `[b, d]`, increment `diff[b - 1950]` and decrement `diff[d - 1950]`.
  - Compute prefix sums. Track the year that achieves the maximum population, returning the earliest year in case of ties.

## Code Implementation
```cpp
class Solution {
public:
    int maximumPopulation(vector<vector<int>>& logs) {
       map<int,int> mp;
       for(auto& it: logs){
            mp[it[0]]++;
            mp[it[1]]--;
       }
       int maxi = 0;
       int year = 0;
       int total = 0;
       for(auto it=mp.begin();it!=mp.end();it++){
            total += it->second;
            if(total > maxi){
                maxi = total;
                year = it->first;
            }
       }
       return year;
        
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N + M)$ where $N$ is the log entries size and $M = 100$ is the years span.
- **Space Complexity**: $O(M)$ auxiliary space.

---
**Key Takeaways**:
- **Temporal Intervals**: Difference arrays track overlapping temporal intervals efficiently by converting range spans into offset index updates.
