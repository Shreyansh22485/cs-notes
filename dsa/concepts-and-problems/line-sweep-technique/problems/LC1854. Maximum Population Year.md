```cpp
class Solution {
public:
    int maximumPopulation(vector<vector<int>>& logs) {
       map<int,int> mp;
       for(auto& it: logs){
            mp[it[0]]++;
            mp[it[1]]--;
       }
       int maxi = 0;
       int year = 0;
       int total = 0;
       for(auto it=mp.begin();it!=mp.end();it++){
            total += it->second;
            if(total > maxi){
                maxi = total;
                year = it->first;
            }
       }
       return year;
        
    }
};
```
- **Event** - birth and death years of the population
- **Sort Events** - sorted by the sweeping dimension (x-coordinates)
- **Sweep and Process** - move the line and update the answer based on event

## Time and Space Complexity
- **Time Complexity** - O(N*logN) due to sorting.
- **Space Complexity** - O(N) for storing the events.
