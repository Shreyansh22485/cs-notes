# LC743. Network Delay Time

[LeetCode Problem Link](https://leetcode.com/problems/network-delay-time/)

## Intuition & Approach
- The problem asks for the minimum time required for all nodes to receive a signal from a source node $K$ in a directed graph with positive edge weights representing propagation delay.
- Since all weights are non-negative, this is a classic Single-Source Shortest Path (SSSP) problem solvable via Dijkstra's algorithm.
- We initialize a `dist` array of size $N$ to infinity, with `dist[K] = 0`.
- We use a min-priority queue to greedily explore the node with the shortest path. For each popped node, we relax all its outgoing edges: `dist[v] = min(dist[v], dist[u] + weight)`.
- After visiting all reachable nodes, the total delay is the maximum value in the `dist` array. If any node is unreachable (distance remains infinity), we return `-1`.

## Code Implementation
```cpp
class Solution {
public:
    
    int networkDelayTime(vector<vector<int>>& edges, int n, int k) {
    vector<pair<int,int>> adj[n+1];
    for(auto it: edges){
        int u=it[0];
        int v=it[1];
        int wt=it[2];
        adj[u].push_back({v,wt});
    }
    vector<int> dist(n+1,1e8);
    priority_queue<pair<int,int>,vector<pair<int,int>>,greater<pair<int,int>>> pq;
    pq.push({0,k});
    dist[k]=0;

    while(!pq.empty()){
        auto it=pq.top();pq.pop();
        int node=it.second;
        int d=it.first;

        for(auto it:adj[node]){
            int v=it.first;
            int w=it.second;

            if(d+w<dist[v]){
                dist[v]=d+w;
                pq.push({d+w,v});
            }
        }
    }
    int maxi=0;
    for(int i=1;i<=n;i++){
        if(dist[i]==1e8) return -1;
        else maxi=max(maxi,dist[i]);
    }
    return maxi;

    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(E \log V)$ using a binary heap priority queue to extract minimum distances.
- **Space Complexity**: $O(V + E)$ for the adjacency list representation and priority queue elements.

---
**Key Takeaways**:
- **Dijkstra SSSP Standard**: The standard greedy approach for single-source shortest paths on non-negative weighted graphs.
