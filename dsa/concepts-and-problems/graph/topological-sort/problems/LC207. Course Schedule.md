# LC207. Course Schedule

[LeetCode Problem Link](https://leetcode.com/problems/course-schedule/)

## Intuition & Approach
- The problem asks if we can finish all courses given prerequisite pairings. This is equivalent to checking if the dependency graph of courses is acyclic (contains no directed cycles).
- We represent courses as a directed graph where an edge $u \to v$ means course $u$ is a prerequisite for course $v$.
- We can detect cycles using Kahn's algorithm (indegree-based BFS topological sort):
  - Calculate the indegree of all vertices.
  - Queue all vertices with indegree 0 (no prerequisites).
  - Pop a vertex, decrement the indegree of all its neighbors, and push any neighbor that hits indegree 0.
  - Keep a count of visited vertices. If the final count equals the total courses, it is a Directed Acyclic Graph (DAG) and we can complete all courses. Otherwise, a cycle exists.

## Code Implementation
```cpp
class Solution {
public:
    bool canFinish(int n, vector<vector<int>>& pre) {
        vector<int> adj[n];
        vector<int> indegree(n,0);
        for(auto it: pre){
            adj[it[1]].push_back(it[0]);
            indegree[it[0]]++;
        }
        queue<int> q;
        for(int i=0;i<n;i++){
            if(indegree[i]==0){
                q.push(i);
            }
        }
        while(!q.empty()){
            int u=q.front();
            q.pop();
            for(auto v:adj[u]){
                indegree[v]--;
                if(indegree[v]==0){
                    q.push(v);
                }
            }
        }
        for(int i=0;i<n;i++){
            if(indegree[i]!=0){
                return false;
            }
        }
        return true;

    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(V + E)$ where $V = N$ (courses) and $E$ is prerequisites. We process each vertex and edge once.
- **Space Complexity**: $O(V + E)$ for the adjacency list representation, indegree tracker, and queue.

---
**Key Takeaways**:
- **Kahn's Cycle Detection**: If a topological sort queue cannot visit all vertices in a directed graph, the graph contains a directed cycle.
