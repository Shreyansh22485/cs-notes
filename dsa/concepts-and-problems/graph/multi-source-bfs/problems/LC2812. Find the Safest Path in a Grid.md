# LC2812. Find the Safest Path in a Grid

[LeetCode Problem Link](https://leetcode.com/problems/find-the-safest-path-in-a-grid/)

## Intuition & Approach
- The problem asks us to find a path from $(0,0)$ to $(n-1,m-1)$ such that the minimum distance to any thief along the path is maximized (the safeness factor of the path).
- This is a maximin path problem. We can observe that if a path exists with a minimum safeness factor of $X$, a path also exists for any factor $X' < X$. This monotonic property permits Binary Search on the Answer.
- Step 1: Precompute the minimum distance to the nearest thief for every cell. We do this by launching a Multi-Source BFS starting from all thief locations simultaneously, creating a distance matrix `dist`.
- Step 2: Binary search for the maximum safeness factor $X$. For a chosen mid value:
  - Verify if a path exists from $(0,0)$ to $(n-1,m-1)$ using only cells where `dist[i][j] >= mid`.
  - We run a standard BFS/DFS starting at $(0,0)$ to check reachability under this constraint.
  - If reachable, search the right half ($[mid + 1, r]$). Else, search the left half ($[l, mid - 1]$).

## Code Implementation
```cpp
class Solution {
public:
    vector<vector<int>> directions = {{-1,0},{1,0},{0,1},{0,-1}};
    int n;
    int m;

    bool isPossible(int safeFact, vector<vector<int>>& dist){
        if(safeFact > dist[0][0] || safeFact > dist[n-1][m-1]) return false;

        queue<pair<int,int>>q;
        vector<vector<bool>> vis(n,vector<bool>(m,false));

        q.push({0,0});
        vis[0][0] = true;

        while(!q.empty()){
            auto it = q.front();
            q.pop();

            int i = it.first;
            int j = it.second;

            if(i == n-1 && j==m-1) return true;

            for(auto& dir : directions){
                int x = i + dir[0];
                int y = j + dir[1];

                if(x>=0 && x<n && y>=0 && y<m && !vis[x][y] && dist[x][y] >= safeFact){
                    q.push({x,y});
                    vis[x][y] = 1;
                }
            }
        }

        return false;
    }

    int maximumSafenessFactor(vector<vector<int>>& grid) {
        n = grid.size();
        m = grid[0].size();

        
        queue<pair<int,int>> q;
        vector<vector<int>> dist(n,vector<int>(m,-1));

        for(int i=0;i<n;i++){
            for(int j=0;j<m;j++){
                if(grid[i][j] == 1){
                    q.push({i,j});
                    dist[i][j] = 0;
                }
            }
        }

        int level = 0;
        while(!q.empty()){
            int size = q.size();
            while(size--){
                auto it = q.front();
                q.pop();

                int i = it.first;
                int j = it.second;

                for(auto& dir: directions){
                    int x = i + dir[0];
                    int y = j + dir[1];

                    if(x >=0 && x < n && y >=0 && y < m && dist[x][y] == -1){
                        q.push({x,y});
                        dist[x][y] = level+1;
                    }
                }
            }
            if(!q.empty())
                level++;
        }

        int l = 0;
        int r = level;
        int ans = -1;
        while(l <= r){
            int mid = r - (r-l)/2;
            
            if(isPossible(mid,dist)){
                ans = mid;
                l = mid + 1;
            }
            else{
                r = mid - 1;
            }
        }
        
        
        return ans;
    }
};
```

## Complexity Analysis
- **Time Complexity**: 
  - Multi-source BFS: $O(N 	imes M)$
  - Binary Search: $O(\log(\min(N, M)))$ iterations, where each checks reachability in $O(N 	imes M)$ BFS time.
  - Total: $O(N 	imes M \log(\min(N, M)))$.
- **Space Complexity**: $O(N 	imes M)$ to store the distance matrix and queue elements during BFS.

---
**Key Takeaways**:
- **Maximin / Minimax Optimization**: When searching for an optimal limit along a path, binary searching the threshold reduces the problem to reachability verification.
- **Combined BFS and BS**: Precalculating distance maps via BFS before executing binary search is a robust hybrid grid approach.
