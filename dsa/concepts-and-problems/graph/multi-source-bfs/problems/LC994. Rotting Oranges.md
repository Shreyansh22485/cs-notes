# LC994. Rotting Oranges

[LeetCode Problem Link](https://leetcode.com/problems/rotting-oranges/)

## Intuition & Approach
- The problem asks for the minimum minutes to rot all fresh oranges in a grid. Spreading occurs 4-directionally from rotten oranges to fresh oranges. If any fresh orange is unreachable, return -1.
- Spreading processes can be modeled as a level-order traversal (BFS) starting from multiple source nodes (all initial rotten oranges).
- We initialize the BFS queue with coordinates of all cells with value `2` (rotten), and count the initial number of fresh oranges (`total`).
- During BFS:
  - We traverse the queue layer-by-layer (each layer corresponds to 1 minute).
  - For each cell popped, check its neighbors. If a neighbor is a fresh orange, change its state, increment the rotted orange count (`cnt`), and push it to the queue.
  - If we successfully rot at least one orange in the current minute, we increment the `time` counter.
- Finally, if `cnt == total`, return `time`. Otherwise, return `-1`.

## Code Implementation
```cpp
class Solution {
public:
    int orangesRotting(vector<vector<int>>& grid) {
        int m=grid.size();
        int n=grid[0].size();
        queue<pair<int,int>> q;
        vector<vector<int>> vis(m,vector<int>(n,0));
        int total=0;
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                if(grid[i][j]==2){
                    q.push({i,j});
                    vis[i][j]=2;
                }
                else{
                    if(grid[i][j]==1)
                    total++;
                }
            }
        }
        int cnt=0;
        int time=0;
        int dx[]={-1,0,1,0};
        int dy[]={0,1,0,-1};
        while(!q.empty()){
            int size = q.size();
            while(size--){
                auto it=q.front();
                q.pop();
                int row=it.first;
                int col=it.second;

                for(int k=0;k<4;k++){
                    int nrow=row+dx[k];
                    int ncol=col+dy[k];

                    if(nrow>=0 && nrow<m && ncol>=0 && ncol<n && vis[nrow][ncol]==0 && grid[nrow][ncol]==1){
                        q.push({nrow,ncol});
                        vis[nrow][ncol]=2;
                        cnt++;
                    }
                }
            }
            if(!q.empty()) time++;
            
        }
        if(cnt!=total) return -1;
        return time;
        
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N 	imes M)$, where $N$ and $M$ are grid rows and columns. Every cell is visited at most once.
- **Space Complexity**: $O(N 	imes M)$ for the queue and visited matrix.

---
**Key Takeaways**:
- **Simultaneous Expansion**: Multi-source BFS guarantees that the spread expands simultaneously from all sources, calculating the minimal global time.
- **Disconnected Checking**: Comparing final infected count (`cnt`) with initial fresh count (`total`) handles disconnected components efficiently.
