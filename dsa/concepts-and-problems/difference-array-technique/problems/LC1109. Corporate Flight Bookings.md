# LC1109. Corporate Flight Bookings

[LeetCode Problem Link](https://leetcode.com/problems/corporate-flight-bookings/)

## Intuition & Approach
- We have $N$ flights and booking entries of the form `[first, last, seats]`. We need to return the total seats booked for each flight.
- A naive range addition for each booking takes $O(N \times B)$ time. Instead, we use the 1D Difference Array technique to record updates in $O(1)$ time:
  - Create a difference array `diff` of size $N$ initialized to 0.
  - For each booking `[L, R, val]`, add `val` to `diff[L - 1]` and subtract `val` from `diff[R]` (if $R < N$).
  - After recording all bookings, compute the running prefix sum of `diff` to obtain the final seat counts in a single pass.

## Code Implementation
```cpp
class Solution {
public:
    vector<int> corpFlightBookings(vector<vector<int>>& bookings, int n) {
        vector<int>diff(n);
        for(auto& booking: bookings){
            int first = booking[0]-1;
            int last = booking[1]-1;
            int seats = booking[2];

            diff[first] += seats;
            if(last+1 < n)
                diff[last+1] -= seats;

        }

        vector<int> ans(n);
        int totalSum = 0;
        ans[0] = diff[0];
        for(int i=1;i<n;i++){
            diff[i] += diff[i-1];
            ans[i]=diff[i];
        }
        return ans;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N + B)$ where $N$ is flights count and $B$ is bookings size.
- **Space Complexity**: $O(N)$ auxiliary space for the difference array.

---
**Key Takeaways**:
- **1D Range to Point Updates**: Range updates of $[L, R]$ with value $val$ are mapped to `diff[L] += val` and `diff[R+1] -= val` to achieve $O(1)$ query updates.
