```cpp
class Solution {
public:
    int numberOfPoints(vector<vector<int>>& nums) {
        vector<int> mp(102,0);
        for(auto& num : nums){
            mp[num[0]]++;
            mp[num[1]+1]--;
        }
        int ans = 0;
        int sum = 0;
        for(int i =1;i<=100;i++){
            sum+=mp[i];
            if(sum>0) ans++;
        }
        return ans;
    }
};
```
- **Time Complexity** - O(N + M) where N is the number of intervals and M is the range of the numbers (100 in this case).
- **Space Complexity** - O(M) for the difference array.
