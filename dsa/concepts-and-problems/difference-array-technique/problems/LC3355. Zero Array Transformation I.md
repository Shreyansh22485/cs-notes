# LC3355. Zero Array Transformation I

[LeetCode Problem Link](https://leetcode.com/problems/zero-array-transformation-i/)

## Intuition & Approach
- We are given `nums` and queries where each query `[l, r]` can decrement all elements in that range by at most 1. We want to check if `nums` can be converted to all zeros.
- This is equivalent to checking if the total decrement capacity at each index $i$ is at least `nums[i]`.
- We use a difference array of size $N$ to accumulate decrement capabilities:
  - For each query `[l, r]`, perform `diff[l] += 1` and `diff[r + 1] -= 1`.
  - Compute the prefix sum of `diff` to get the total capacity at each index.
  - If `capacity[i] < nums[i]` for any index, return `false`. Else, return `true`.

## Code Implementation
```cpp
class Solution {
public:
    bool isZeroArray(vector<int>& nums, vector<vector<int>>& queries) {
        int n = nums.size();
        vector<int> changes(n,0);
        for(auto& query : queries){
            int l = query[0];
            int r = query[1];

            changes[l] += -1;
            if(r+1<n) changes[r+1] -= -1;
        }
        for(int i=1;i<n;i++){
            changes[i] = changes[i] + changes[i-1];
        }

        for(int i=0;i<n;i++){
            if(nums[i] + changes[i] > 0) return false;
        }
        return true;
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N + Q)$ where $N$ is array size and $Q$ is queries size.
- **Space Complexity**: $O(N)$ for the difference array.

---
**Key Takeaways**:
- **Capacity Validation Pattern**: A difference array aggregates capacity from range queries to verify index demands in linear time.
