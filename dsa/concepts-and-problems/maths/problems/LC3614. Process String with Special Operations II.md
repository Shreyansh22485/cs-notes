# LC3614. Process String with Special Operations II

[LeetCode Problem Link](https://leetcode.com/problems/process-string-with-special-operations-ii/)

## Intuition & Approach
- We are given a string `s` containing characters, `*`, `#`, and `%`, representing operations that modify string length:
  - Letters insert a character (length increases by 1).
  - `*` deletes the last character (length decreases by 1).
  - `#` duplicates the string (length doubles).
  - `%` reverses the string.
- We want to find the character at index `k` in the final string.
- A forward simulation is impossible because the string size doubles exponentially with `#`, exceeding memory limits.
- We perform **Reverse Simulation** instead:
  - First, run a forward pass keeping track of the string length `L` at each step without actually building the string.
  - Then, run a backward pass starting from the end of the string.
  - As we traverse backward:
    - If the character is `#`, the string was duplicated. If the query index $k \ge L / 2$, then $k$ maps to $k \pmod{L/2}$ in the original half. Update $k = k \pmod{L/2}$ and $L = L / 2$.
    - If the character is `%`, the string was reversed, so $k$ maps to $L - k - 1$.
    - If the character is a letter, and the current length $L$ equals $k + 1$ (0-indexed), then this is the target character we are searching for, so we return it!
    - If the character is `*`, the length was smaller before, so we adjust $L = L + 1$.

## Code Implementation
```cpp
class Solution {
public:
    char processStr(string s, long long k) {
        long long L = 0;

        // Simulation to Find L

        for(int i=0;i<s.size();i++){
            char ch = s[i];
            if(ch >= 'a' && ch <= 'z'){
                L++;
            }
            else if(ch == '*'){
                if(L>0)
                    L--;
            }
            else if(ch == '#'){
                L *= 2;
            }
        }
        if(k>=L) return '.';

        // Reverse Simulating
        

        for(int i=s.size()-1;i>=0;i--){
            char ch = s[i];
            if(ch >= 'a' && ch <='z'){
                L--;
                if(L==k){
                    return ch;
                }
            }
            else if(ch == '*'){
                L++;
            }
            else if(ch == '#'){
                L = L/2;
                k=k%L;
            }
            else if(ch == '%'){
                k = L - k - 1;
            }
        }

        return '.';
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N)$ with a forward pass and a backward pass over the operations string.
- **Space Complexity**: $O(1)$ auxiliary space.

---
**Key Takeaways**:
- **Reverse Index Mapping**: When a forward process expands sizes exponentially, track the sizes forward and map the target index backward recursively or iteratively.
