# LC142. Linked List Cycle II

[LeetCode Problem Link](https://leetcode.com/problems/linked-list-cycle-ii/)

## Intuition & Approach
- We want to find the exact node where the cycle begins.
- First, we detect the cycle using the fast-slow pointer approach. Let the meeting node be `meet`.
- Mathematically, if $L_1$ is the distance from the head to the cycle start, and $L_2$ is the distance from the cycle start to the meeting point, the distance traveled by `slow` is $L_1 + L_2$, and by `fast` is $L_1 + L_2 + c \cdot C$ (where $C$ is cycle length).
- Since `fast` travels twice as fast as `slow`: $2(L_1 + L_2) = L_1 + L_2 + c \cdot C \implies L_1 = c \cdot C - L_2$.
- This implies that the distance from the head to the cycle start ($L_1$) is equal to the distance from the meeting point to the cycle start going forward.
- Therefore, we reset `slow` to `head` and keep `fast` at the meeting point. We then move both pointers one step at a time. The node where they meet is the start of the cycle.

## Code Implementation
```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
        ListNode* slow=head;
        ListNode* fast=head;
        while(fast && fast->next){
            slow=slow->next;
            fast=fast->next->next;
            if(slow==fast){
                slow=head;
                int length=0;
                while(slow!=fast){
                    slow=slow->next;
                    fast=fast->next;
                    length++;
                }
                cout<<"Length of the cycle is: "<<length<<endl;
                return slow;
            }
        }
        return NULL;
        
    }
};
```

## Complexity Analysis
- **Time Complexity**: $O(N)$ time.
- **Space Complexity**: $O(1)$ auxiliary space.

---
**Key Takeaways**:
- **Cycle Entry Point Derivation**: Resetting one pointer to the head after a cycle meeting and moving both at equal speed identifies the cycle entry point.
