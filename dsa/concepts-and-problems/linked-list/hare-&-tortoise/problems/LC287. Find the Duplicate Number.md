```cpp
class Solution {
public:
    int findDuplicate(vector<int>& nums) {
        int n=nums.size();
        int slow=nums[0];
        int fast=nums[0];
        do{
            slow=nums[slow];
            fast=nums[nums[fast]];
        }while(slow!=fast);
        fast=nums[0];
        while(slow!=fast){
            slow=nums[slow];
            fast=nums[fast];
        }
        return slow;
    }
};
```
- Time Complexity: O(n), where n is the length of the input array nums. The algorithm traverses the array a few times, but each traversal is linear.
- Space Complexity: O(1), as we are using only a constant amount of extra space for the two pointers (slow and fast). The input array is modified in place, but we are not using any additional data structures that grow with the input size.
