# Monotonic Data Structures

## Monotonic Increasing
- A data structure is said to be monotonic increasing if the elements are in non-decreasing order. In other words, each element is greater than or equal to the previous element.

### Monotonic Decreasing
- A data structure is said to be monotonic decreasing if the elements are in non-increasing order. In other words, each element is less than or equal to the previous element.

# Why Monotonic Data Structures?

- Next greater/smaller element problems
- Sliding window minimum/maximum problems
- Finding min/max in a dynamic ranges
- Stock span problems
